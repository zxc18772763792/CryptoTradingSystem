1. 项目概述
================================================================================

1.1 目标
--------
为 crypto_trading_system 添加以下免费数据源支持:
- 资金费率 (Funding Rate) - 4大交易所
- 恐惧贪婪指数 (Fear & Greed Index)
- 订单簿 Level 2 数据
- 未平仓合约 (Open Interest)
- 链上基础数据 (Etherscan, Dune)
- 社交情绪 (Reddit, CryptoCompare)

1.2 现有架构
------------
- core/data/data_collector.py - 数据采集基类
- core/exchanges/ - 交易所连接器 (Binance, Bybit, OKX, Gate)
- core/factors_ts/ - 因子计算框架
- core/marketdata/ - WebSocket 客户端框架

1.3 扩展原则
------------
- 所有数据源必须免费
- 遵循现有代码风格
- 使用 async/await 模式
- 添加完整的日志记录
- 编写单元测试

class FundingRateCollector:
    - __init__():

        参数: category=linear, symbol
    
    - async fetch_okx(symbol: str) -> FundingRate:
        GET https://www.okx.com/api/v5/public/funding-rate
        参数: instId
    
    - async fetch_gate(symbol: str) -> FundingRate:
        GET https://api.gateio.ws/api/v4/futures/usdt/funding_rate
        参数: contract
    
    - async fetch_all(symbol: str) -> Dict[str, FundingRate]:
        并行获取所有交易所数据
    
    - start_collection(symbols: List[str], interval: int = 60):
        启动定时采集

Step 3: 创建资金费率因子
文件: core/factors_ts/funding_rate_factors.py

class FundingRateFactor(TimeSeriesFactor):
    - 当前资金费率
    
class FundingRateZscoreFactor(TimeSeriesFactor):
    - 资金费率 Z-score (历史标准化)
    - 窗口: 30/60/90 期
    
class FundingRateMomentumFactor(TimeSeriesFactor):
    - 资金费率变化趋势
    - 计算: rate[t] - rate[t-n]
    
class FundingRateExtremeFactor(TimeSeriesFactor):
    - 极端资金费率检测
    - 阈值: > 0.01 或 < -0.01
    
class BasisFactor(TimeSeriesFactor):
    - 现货-期货价差
    - 需要: 现货价格 + 永续价格 + 资金费率

class OrderbookWSClient:

Step 2: 创建订单簿数据模型
文件: core/marketdata/orderbook_models.py

@dataclass
class PriceLevel:
    price: float
    size: float
    
@dataclass
class OrderBook:
    exchange: str
    symbol: str
    bids: List[PriceLevel]  # 买单 (价格降序)
    asks: List[PriceLevel]  # 卖单 (价格升序)
    timestamp: datetime
    
    - mid_price() -> float:
        中间价
    
    - spread() -> float:
        买卖价差
    
    - bid_notional() -> float:
        买盘总价值
    
    - ask_notional() -> float:
        卖盘总价值
    
    - imbalance() -> float:
        买卖失衡 (-1 到 1)

Step 3: 创建订单簿因子
文件: core/factors_ts/orderbook_factors.py

class RealKyleLambdaFactor(TimeSeriesFactor):
    """真实 Kyle's Lambda - 价格冲击系数"""
    - 计算方法: 回归 |价格变化| ~ |净订单流|
    
class VPINFactor(TimeSeriesFactor):
    """VPIN - Volume-synchronized Probability of Informed Trading"""
    - 参考: Easley et al. (2012)
    
class BidAskImbalanceFactor(TimeSeriesFactor):
    """买卖盘失衡"""
    - 计算: (bid_notional - ask_notional) / (bid_notional + ask_notional)
    
class DepthSkewFactor(TimeSeriesFactor):
    """深度倾斜"""
    - 计算: log(bid_depth / ask_depth)
    
class SpreadFactor(TimeSeriesFactor):
    """买卖价差"""
    - 绝对价差
    - 相对价差 (bps)
    
class WeightedMidPriceFactor(TimeSeriesFactor):
    """加权中间价偏离"""
    - 使用订单量加权的中间价

4.3 订单簿处理逻辑
------------------

1. 增量更新:
   - Binance: 发送增量更新，需要本地维护完整订单簿
   - Bybit: 发送快照+增量
   - OKX: 发送全量快照

2. 深度管理:
   - 维护固定深度 (如 50 档)
   - 定期清理过期价格档位
   - 处理价格精度

3. 重连机制:
   - 检测连接断开
   - 自动重连并重新订阅
   - 重新获取快照

6.1.1 Etherscan (免费层)
------------------------

- ERC20 转账: ?module=account&action=tokentx&address={address}
- Gas 价格: ?module=gastracker&action=gasoracle

- 稳定币市值变化
- DeFi 协议 TVL
- NFT 交易量

6.1.3 Arkham (免费层)

```python
import asyncio

async def test_funding_rate():
    from core.data.funding_rate_collector import FundingRateCollector
    collector = FundingRateCollector()
    
    # 测试 Binance
    rate = await collector.fetch_binance("BTCUSDT")
    print(f"Binance BTCUSDT funding rate: {rate.funding_rate}")
    
    # 测试所有交易所
    rates = await collector.fetch_all("BTCUSDT")
    for ex, r in rates.items():
        print(f"{ex}: {r.funding_rate}")

async def test_fear_greed():
    from core.data.sentiment.fear_greed_collector import FearGreedCollector
    collector = FearGreedCollector()
    
    index = await collector.fetch_current()
    print(f"Fear & Greed Index: {index.value} ({index.value_classification})")

async def test_orderbook():
    from core.marketdata.orderbook_ws_client import OrderbookWSClient
    client = OrderbookWSClient()
    
    await client.subscribe_binance("BTCUSDT")
    await asyncio.sleep(2)
    
    ob = client.get_orderbook("binance", "BTCUSDT")
    print(f"Best bid: {ob.bids[0].price}")
    print(f"Best ask: {ob.asks[0].price}")
    print(f"Spread: {ob.spread()}")

if __name__ == "__main__":
    asyncio.run(test_funding_rate())
    asyncio.run(test_fear_greed())
    asyncio.run(test_orderbook())
```
